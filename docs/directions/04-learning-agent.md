# 方向四：技术学习 Agent — 执行方案

## 产品定位

**"把碎片化的技术探索变成系统性知识"** — 专门帮技术人员学习新技术、管理学习进度、构建个人技术知识体系。

不是又一个 AI 问答工具，而是一个有记忆、懂你技术背景、能主动推进学习的私人技术导师。

---

## 核心价值主张

- 学过的东西不再忘：自动沉淀，随时召回
- 学习路径个性化：根据你的项目和技术栈推荐下一步
- 碎片时间有效利用：5 分钟也能推进一个知识点
- 知识体系可视化：看到自己的技术成长轨迹

---

## 目标用户

- 需要快速上手新技术栈的工程师（换工作、新项目）
- 想系统学习某个领域的自学者（分布式、编译器、ML 等）
- 技术广度不够、想补课的开发者
- 准备技术面试的工程师

---

## 核心功能模块

### 1. 学习记忆系统
- 记录每次学习的内容：问了什么、理解了什么、还有哪些疑问
- 自动识别"已掌握"vs"模糊"vs"未学"的知识点
- 遗忘曲线提醒：根据艾宾浩斯曲线安排复习
- 知识点之间的关联自动建立

### 2. 个性化学习路径
- 输入目标（"我想学 Rust 异步编程"）
- Agent 根据你的现有知识背景生成学习路径
- 路径动态调整：根据学习进度和理解情况实时更新
- 关联你的项目上下文（"你在用 Tokio，先学这个"）

### 3. 知识库管理
- 导入学习资料：文章 URL、PDF、视频字幕、书籍章节
- 自动提取关键概念和知识点
- 与已有知识关联，避免重复学习
- 标注来源，方便回溯

### 4. 主动学习 Agent
- 苏格拉底式提问：不直接给答案，引导你推导
- 费曼技巧：让你用自己的话解释，检验理解深度
- 生成练习题（概念题 + 编程题）
- 代码实验：直接在 agent 里跑代码验证理解

### 5. 知识图谱
- 可视化你的技术知识体系
- 识别知识盲区（"你对 X 了解很少，但它是 Y 的基础"）
- 跨领域关联（"你学的 Rust 所有权和 GC 原理有这些联系"）

### 6. 学习报告
- 每周学习总结：学了什么、掌握程度、时间投入
- 知识增长曲线
- 推荐下周学习重点

## 技术架构

```
输入层:
  - CLI（问答、记录）
  - URL/文件导入
  - 代码执行环境（学习验证）

学习引擎:
  - KnowledgeTracker：知识点状态追踪
  - PathPlanner：学习路径规划
  - ReviewScheduler：复习调度（遗忘曲线）
  - SocraticAgent：引导式问答

知识库:
  - 知识点图谱（概念 + 关系）
  - 学习资料索引（向量检索）
  - 个人笔记（结构化 + 向量）

记忆层:
  - 学习历史（SQLite）
  - 理解状态（每个知识点的掌握程度）
  - 复习计划（调度队列）

输出层:
  - 交互式学习会话
  - 学习报告（Markdown）
  - 知识图谱导出
```

---

## MVP 功能范围（6-8 周）

**Week 1-2：基础问答 + 记忆**
- LLM 接入
- 学习记录存储（问了什么、学了什么）
- 基础知识点提取

**Week 3-4：学习路径**
- 目标输入 → 路径生成
- 知识点状态追踪（已学/未学/模糊）
- 进度可视化（CLI 文本形式）

**Week 5-6：知识库**
- URL/文件导入 + 向量索引
- 知识关联建立
- 相关资料推荐

**Week 7-8：主动学习**
- 遗忘曲线复习提醒
- 苏格拉底式提问模式
- 每周学习报告生成

---

## 差异化壁垒

1. **个人知识图谱**：随时间积累，越用越懂你，竞品无法复制
2. **项目上下文关联**：根据你实际在做的项目推荐学习内容，不是通用课程
3. **本地隐私**：学习记录、笔记、资料不上传
4. **主动性**：不只是被动回答，会主动提醒复习、推进学习进度

---

## 与现有工具的区别

| | 本产品 | ChatGPT/Claude | Anki | 在线课程 |
|---|---|---|---|---|
| 记忆 | 持久个人知识图谱 | 无 | 卡片记忆 | 无 |
| 个性化 | 基于你的项目和背景 | 无 | 无 | 弱 |
| 主动性 | 主动推进 + 提醒 | 被动回答 | 被动 | 固定进度 |
| 代码实验 | 内置执行环境 | 无 | 无 | 部分有 |
| 本地 | 完全本地 | 云端 | 本地 | 云端 |

## 商业化路径

- 个人版：免费（基础问答 + 记忆）
- Pro 版：完整知识图谱 + 学习路径 + 报告（$10/月）
- 团队版：团队知识库共享 + 新人培训路径（$8/人/月）

---

## 风险点

- 学习类产品用户留存难，需要强习惯养成机制
- 知识点提取和关联的准确性依赖 LLM 质量
- 遗忘曲线调度需要用户持续使用才有效，冷启动体验差
- 竞品：Notion AI、Obsidian + AI 插件有类似功能，需要明确差异
